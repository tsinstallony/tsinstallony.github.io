Git上传项目到远程仓库

十一月 01, 2019
1.安装git软件： 这个git-scm.com
2.建立Git与Github的远程联系
  • 打开你的Github点击头像下的Setting（设置）
  • 找到个人设置下的SSH and GPG keys（SSH和GPG密钥）
  • 点击New SSH key （新的SSH密钥）
  • 然后打开本地文件夹C:\Users\Wangly \ .ssh下的id_[rsa.pub]将里面的内容复制到刚才打开的Github新的SSH key里面保存，没有文件的可以用 $ ssh-keygen -t rsa -C “[youremail@email.com]”初始化
  • 以上就是建立SSH连接
3.初始化本地仓库
  • 新建一个文件夹
  • 右键点击Git Bash Here进入到Git命令窗口
  • 初始化完成后将项目文件放进去
  • 添加全部文件到git
  • 把项目提交到仓库中
  • 这样就关联好你的远程仓库了
  • 如果你远程仓库是空的话需要加上 -u 哦，下一次就不需要了
  • 如果有上面的错误那么代表你的github仓库有说明文档，默认创建的README文件
  • 合并一下重新输入上面的git push -u origin master代码就好了
